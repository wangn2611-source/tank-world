# 坦克大战 - Battle City 🎮

经典坦克大战游戏的 HTML5 版本，支持电脑端游玩。

## 🎯 游戏说明

- **目标**：消灭所有敌方坦克，保护我方基地（左下角房屋）
- **关卡**：每关 20 个敌人，全部消灭进入下一关
- **生命**：初始 3 条命，被击中后会在基地附近重生

## 🎮 操作控制

| 操作 | 按键 |
|------|------|
| 移动 | `↑ ↓ ← →` 或 `W A S D` |
| 射击 | `空格` 或 `J` |
| 开始/继续 | `Enter` |
| 暂停 | `P` |
| 重新开始 | 点击右侧按钮 |

## ✨ 特性

- 砖墙、钢墙、水域等地形
- 敌方 AI 坦克自动寻敌
- 爆炸特效
- 道具系统（加速、护盾）
- 音效（Web Audio）
- 计分系统

## 🚀 部署

直接打开 `index.html` 即可游玩。

或访问 GitHub Pages 在线游玩：`https://你的用户名.github.io/仓库名/`
